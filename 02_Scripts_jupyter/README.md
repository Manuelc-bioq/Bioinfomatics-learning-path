# 🐍 02: Python Scripts & Data Analysis

## 📌 About this folder
This directory contains all my Jupyter Notebooks (`.ipynb`) and Python scripts developed for bioinformatics data analysis. It serves as a central hub for my computational practice.

## 🗂️ Topics Covered
- **Data Manipulation:** Reading, filtering, and processing `.csv` and `.tsv` biological datasets using `pandas`.
- **Sequence Analysis:** Handling and parsing `.fasta` files.
- **Structural Biology:** Parsing `.pdb` files, extracting 3D atomic coordinates, and navigating the SMCRA hierarchy using `BioPython`.

## 💻 Tech Stack
- Python 3
- Libraries: `pandas`, `BioPython`, `urllib`
- Environment: Jupyter Notebook / Conda
