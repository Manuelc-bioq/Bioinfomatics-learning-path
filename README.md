# 🧬 Bioinformatics & Protein Design: Learning Path

# Documenting my journey learning Bioinformatics, Python, and Protein Design from scratch.

## 📌 About this repository
This repository serves as my coding diary and portfolio. It documents my 4-year journey learning Bioinformatics, Structural Biology, and Protein Design from scratch. 

Rather than polished final software, this is a collection of scripts, data analysis exercises, and molecular visualizations that track my learning curve and practical understanding of biological data.

## 🗂️ Repository Structure
* **`01_PyMOL_Visualizations/`**: 3D structural analysis, binding pockets, and molecular interactions.
* **`02_Python_PDB_Parsing/`**: Data extraction and manipulation of `.pdb` files using Python and BioPython.

## 💻 Tech Stack & Tools
- Python 3 (`pandas`, `BioPython`)
- PyMOL
- Jupyter Notebooks / Conda environments
